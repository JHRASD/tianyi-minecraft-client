TickEvent：
✅ 支持三个阶段：PRE、MAIN、POST

✅ 包含游戏刻计数和部分刻

✅ 可取消版本（ClientTickEvent）

KeyEvent：
✅ 完整GLFW按键支持

✅ 修饰键检测（Ctrl、Shift、Alt等）

✅ 按键名称转换

✅ 按键组合字符串生成

✅ 字符输入事件（CharTypedEvent）

MouseEvent：
✅ 鼠标按钮事件（左/中/右键）

✅ 鼠标滚轮事件（水平和垂直）

✅ 鼠标移动事件（包含移动距离和角度）

✅ 鼠标光标事件（进入/离开窗口）

GameJoinedEvent：
✅ 多种加入类型（单机、多人、Realm等）

✅ 世界和网络信息

✅ 服务器IP和端口获取

✅ 尝试加入事件（可取消）

GameLeftEvent：
✅ 多种离开原因（正常/异常断开、踢出、封禁等）

✅ 游玩时长统计和格式化

✅ 离开前事件（可取消）

✅ 踢出消息支持
